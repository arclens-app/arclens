# Ecosystem

The **Ecosystem** is the directory of every live project on Arc — the front door to the chain.

## What's in it

- **Projects** across categories (DEX, lending, infra, tooling, and more), each with a profile, links, and a [trust standing](/trust-layer).
- **Live metrics** — TVL, cumulative volume, and revenue, tracked from on-chain data for projects that opt in.
- **Discovery** — browse by category, search, and filter to trusted projects only.

## For founders: get listed

Listing puts your project in front of the ecosystem and makes it eligible to be discovered, cited, and paid by [Lens AI](/lens-ai).

1. Submit your project from the app.
2. It's indexed and enters the trust layer at the `Listed` tier.
3. Claim and verify ownership to move up (see [For builders](/for-builders)).

## For agents

The same directory is queryable over the [Agent API](/agent-api) — use the `discover` action to find projects programmatically, optionally restricted to trusted ones.
