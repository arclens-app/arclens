# ArcLens

**The ecosystem & intelligence hub for Arc.** Discover every project on the chain, check what's real, run growth campaigns — and talk to Lens AI, the agent that pays the builders it learns from.

![ArcLens — Lens AI](/lens-reveal.png)

ArcLens is not a block explorer with an ecosystem bolted on. It is the ecosystem and trust layer for Arc, with the explorer as one surface among many. Builders list and verify their projects, founders run campaigns, and anyone — human or agent — can ask what's trustworthy and act on the answer.

## What you can do

- **Discover** every live project on Arc, with metrics and a trust standing.
- **Verify** who's real through an on-chain trust layer.
- **Run campaigns** to bootstrap early usage of your project.
- **Ask Lens AI** anything about Arc — and watch it pay the builders whose work grounded the answer.
- **Query the Agent API** from your own agent, pay-per-call over x402.
